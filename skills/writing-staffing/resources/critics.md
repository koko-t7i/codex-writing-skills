# Critic Roles

Use `cw-agent-critic` for craft critique, `cw-agent-continuity-checker` for canon contradictions, and `cw-agent-reader-sim` for experiential response.

Useful focus areas: structure, continuity, voice, character motivation, dialogue, pacing, prose texture, genre fit, and hook strength.

For high-stakes scenes, run separate passes for craft, continuity, and reader experience only when the user asks for a fuller workflow.
