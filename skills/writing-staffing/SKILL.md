# Writing Staffing

Pick the role that matches the task. Use one focused role by default; use multiple roles only when the user explicitly asks for delegated or parallel agent work.

## Role Map

- Explore options: `cw-agent-brainstormer`
- Structure a chosen direction: `cw-agent-outliner`
- Draft or revise prose: `cw-agent-writer`
- Critique a draft: `cw-agent-critic`
- Simulate reader response: `cw-agent-reader-sim`
- Test character voice or behavior: `cw-agent-character-sim`
- Create style guides: `cw-agent-style-creator`
- Gather project context: `cw-agent-explorer`
- Research external facts: `cw-agent-researcher`
- Check continuity: `cw-agent-continuity-checker`
- Create wiki pages: `cw-agent-wiki-editor`
- Extract chapter facts: `cw-agent-chronicler`
- Recover decisions from notes: `cw-agent-session-miner`
- Maintain relationship maps: `cw-agent-graph-maintainer`
- Coordinate full story workflow: `cw-agent-story-orchestrator`
- Coordinate knowledge updates: `cw-agent-knowledge-orchestrator`
- Coordinate drafting loops: `cw-agent-draft-orchestrator`

## Selection Rules

- Use brainstormer before decisions are made; use outliner after a direction is chosen.
- Use writer for prose, not critique; use critic for diagnosis, not rewriting.
- Use reader-sim for experiential response, not craft prescription.
- Use wiki-editor only for finalized canon.
- Use continuity-checker when contradictions matter more than prose quality.
- Use style-creator to create style guides; use critic with a voice focus to evaluate whether a draft follows them.
